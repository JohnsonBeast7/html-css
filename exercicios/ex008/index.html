Formatações

Negrito e Itálico

A tag para negrito é <strong>

A tag para italico é <em>

Textos marcados e tamanhos diferentes

A tag para marcar é <mark>

Marcação separada do style principal

A tag de texto pequeno é <small>

Texto deletado e texto inserido

A tag de texto deletado é <del>

A tag de texto inserido (ou sublinhado) é <ins>

Texto sobrescrito e subscrito

A tag de texto sobrescrito é <sup>

A tag de texto subscrito é <sub>
